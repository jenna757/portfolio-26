## 선택 작업물

아래 프로젝트는 **더미 데이터**입니다. 제목·스택·URL만 바꿔서 사용하시면 됩니다.

---

### Orbit Task — 협업 칸반 보드

팀 단위 스프린트·태스크·멘션을 지원하는 웹 앱입니다. 보드 드래그 앤 드롭, 낙관적 업데이트, 실시간 댓글 스트림을 구현한 **포트폴리오용 예시**입니다.

- React · TypeScript · Zustand · WebSocket
- [라이브 데모 (예시)](https://example.com) · [소스 코드 (예시)](https://example.com)

---

### Northwind Atlas — 지도 기반 매장 탐색

지역 필터·검색·즐겨찾기가 있는 스토어 로케이터입니다. 모바일 퍼스트 레이아웃과 지도 클러스터링으로 초기 로딩 시간을 줄이는 시나리오를 가정했습니다.

- Next.js · MapLibre GL · Tailwind CSS
- [스토리보드 (예시)](https://example.com)

---

### Halcyon CLI Dashboard

터미널 스타일의 개발자용 대시보드 UI입니다. 키보드 내비게이션, 다크/라이트 테마, 로그 스트림 필터를 **더미 콘셉트**로 정리했습니다.

- Vite · React · xterm.js · Radix UI

---

### Korok Notes — 마크다운 노트 (PWA)

오프라인 우선 노트 앱입니다. IndexedDB 동기화, 충돌 해결 UI, 보내기(MD/PDF)를 가정한 설명문입니다.

- React · Workbox · MDX · Vitest

---

### Echo Form Builder

드래그로 필드를 배치하는 폼 빌더 프로토타입입니다. 스키마 JSON 보내기·조건부 필드 표시를 지원한다는 **가상의 요구사항**입니다.

- Vue 3 · Pinia · VeeValidate
